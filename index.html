VersionControl_Git by ebihara

VersionControl_Git is a library that provides OO interface to handle Git repository.

Mailing list

Download

You can install this package by using pear command: 

$ pear install VersionControl_Git-0.4.3

You can also clone the project with Git by running: 

$ git clone git://github.com/ebihara/VersionControl_Git
